# VATSA Safety Policy v0.1

## Intended Use
VATSA is a research prototype for academic purposes only.
Not approved for clinical, regulated, or production deployment.

## Prohibited Uses
- Individual identification or surveillance
- Autonomous physical action without human oversight
- Processing of real personal data without consent
- Deployment in safety-critical environments

## Known Limitations
- Trained on synthetic/limited data
- Guardrail implementation in progress
- Action head not validated for real-world use

## Responsible Disclosure
Report safety concerns via GitHub Issues.
